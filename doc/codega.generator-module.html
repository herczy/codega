Generator handling

This module is responsible for defining different generator classes. While the GeneratorBase only supplies the API, further generators have extended functionality.

Special constants: PRI_HIGHEST -- Highest priority PRI_HIGH -- High priority PRI_BASE -- Base priority PRI_LOW -- Low priority PRI_LOWEST -- Lowest priority PRI_FALLBACK -- Fallback priority
